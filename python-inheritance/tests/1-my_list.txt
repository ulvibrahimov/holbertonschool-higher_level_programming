=============================
How to use the MyList class
=============================

This module provides a class `MyList` that inherits from `list`.

Importing the class:
>>> MyList = __import__('1-my_list').MyList

Creating an empty instance:
>>> m = MyList()
>>> m
[]

Printing an empty list:
>>> m.print_sorted()
[]

Appending items to the list:
>>> m.append(1)
>>> m.append(4)
>>> m.append(2)
>>> m.append(3)
>>> m.append(5)
>>> m
[1, 4, 2, 3, 5]

Printing the list sorted (ascending):
>>> m.print_sorted()
[1, 2, 3, 4, 5]

Ensuring the original list remains unsorted:
>>> m
[1, 4, 2, 3, 5]

Testing with negative numbers:
>>> m2 = MyList()
>>> m2.append(10)
>>> m2.append(-5)
>>> m2.append(-12)
>>> m2.append(0)
>>> m2.print_sorted()
[-12, -5, 0, 10]
