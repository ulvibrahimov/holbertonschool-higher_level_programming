======================================
How to Use 5-text_indentation
======================================

First import it:
    >>> text_indentation = __import__('5-text_indentation').text_indentation

1. Normal text:
    >>> text_indentation("Holberton. School? Is: Cool")
    Holberton.
    <BLANKLINE>
    School?
    <BLANKLINE>
    Is:
    <BLANKLINE>
    Cool

2. Text ending with punctuation:
    >>> text_indentation("Hello.")
    Hello.
    <BLANKLINE>

3. Text with lots of spaces:
    >>> text_indentation("   Hello  .  World  ")
    Hello.
    <BLANKLINE>
    World

4. Not a string (int):
    >>> text_indentation(123)
    Traceback (most recent call last):
        ...
    TypeError: text must be a string

5. Not a string (float):
    >>> text_indentation(3.14)
    Traceback (most recent call last):
        ...
    TypeError: text must be a string

6. Empty string:
    >>> text_indentation("")

7. Missing argument:
    >>> text_indentation()
    Traceback (most recent call last):
        ...
    TypeError: text_indentation() missing 1 required positional argument: 'text'

8. Text with consecutive punctuation:
    >>> text_indentation("What?: Yes.")
    What?
    <BLANKLINE>
    :
    <BLANKLINE>
    Yes.
    <BLANKLINE>
