======================================
How to Use 2-matrix_divided
======================================

First import it:
    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided

1. Regular division:
    >>> matrix = [[1, 2, 3], [4, 5, 6]]
    >>> matrix_divided(matrix, 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

2. Division by a float:
    >>> matrix = [[1.2, 2.4], [3.6, 4.8]]
    >>> matrix_divided(matrix, 2)
    [[0.6, 1.2], [1.8, 2.4]]

3. Division by a negative number:
    >>> matrix = [[2, 4], [6, 8]]
    >>> matrix_divided(matrix, -2)
    [[-1.0, -2.0], [-3.0, -4.0]]
